Ini sebenernya cuma web iseng yang Aku bikin karena gemes masih banyak orang yang bilang Matcha itu rasanya kayak rumput (¬_¬ ) apalagi anomali satu ini :>

Jadi, aku bikin web ini buat "maksa" biar dia tobat dan akhirnya ngaku kalau Matcha itu enak. Isinya full interaksi, animasi haKamus, sama sedikit pressure mental.

[ FEATURE LIST ]

1. >> Tema yang Bisa Berubah-ubah

Aku pasang 3 mode tampilan biar gak bosen:

Mode Normal (Sage): Tampilan awal yang kalem, adem, estetik lah pokoknya.

Mode Matcha (Neon): Mode gelap tapi teksnya nyala ijo neon. Udah Aku tweak biar tulisannya tetep jelas walau background-nya gelap.

Mode Bahaya (Danger): Nah ini, kalo Kamu nekat nolak Matcha, layarnya bakal jadi merah + goyang-goyang (ada sirinenya visualnya juga).

2. >> Gacha Minuman (Settingan Bandar)

Ada fitur mesin slot buat milih minuman hari ini.

Curang Dikit: Awalnya sih pilihannya banyak (Kopi, Boba, dll), tapi tenang aja, ending-nya udah Aku setting biar pasti dapetnya Matcha. ( ◡̀_◡́)

Efek Glitch: Pas mesinnya "maksa" ke Matcha, teksnya bakal nge-glitch gitu biar dramatis.

3. >> Pop-up Alert Custom

Aku gak pake alert bawaan browser yang ngebosenin itu.

Bikin Kaget: Pop-up nya munculnya agak "nendang" (bouncy) biar Kamu notice.

S & K Maksa: Pas baru buka, ada syarat ketentuan yang gak ada tombol close-nya. Mau gak mau harus klik "Setuju".

4. >> Survey Jebakan Betmen

Di bawah ada survey kepuasan:

Tombol "Enggak": Jangan dipencet. Kalo nekat, auto masuk Mode Bahaya.

Tombol "Mau Banget": Ini baru muncul kalo Kamu udah tobat dari Mode Bahaya. Ada efek confetti-nya juga biar meriah.

5. >> Detail-detail Kecil (Gimmick)

Kursor Matcha: Kalo klik di mana aja, bakal keKamuar debu-debu ijo.

Parallax (HP): Kalo buka di HP, goyangin dikit HP-nya, nanti background-nya gerak.

Judul Tab Marah: Coba deh pindah tab browser, judulnya bakal berubah jadi "JANGAN KABUR WOI!". (╬ Ò﹏Ó)

Tombol Ripple: Kalo diklik ada efek airnya gitu.
